# coox-app
A social media web app that allows people to share their most exciting cooking experiences.
Built using the MERN framework - MongoDB, Express, React, and Node.
Other technologies to be used include:
--GraphQL
--Websockets - Socket.io
--Cloudinary for remote image management
--Mocha and Jasmine for testing
--AWS S3 and Elastik BeanStalk for hosting
